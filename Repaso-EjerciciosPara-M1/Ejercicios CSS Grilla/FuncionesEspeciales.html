KEYWORDS ESPECIALES

fr : Es una unidad de medida especial de css grid para darle ancho o alto a filas y columnas 1fr representa una fraccion del total de columnas o filas. min-content : Ajusta el ancho de la celda lo mínimo posible sin romper su contenido. max-content : Ajusta el ancho de la celda lo máximo posible para mostrar su contenido. auto-fill : Agrega columnas “fantasma” que rellenan el espacio sobrante del contenedor. auto-fit : Ensancha las columnas para que ocupen todo el espacio del contenedor. auto-fill y auto-fit ayudan a la grilla a ocupar el 100% del espacio disponible.
